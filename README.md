## EV Analytics Dashboard

### Features
- CSV-based real-world data ingestion
- KPI metrics derived from raw data
- Interactive charts using Recharts
- Responsive dashboard layout

### Tech Stack
- Next.js (App Router)
- Tailwind CSS
- Recharts
- PapaParse

### Key Insights
- Tesla dominates EV market share
- BEVs account for majority adoption
- Strong post-2020 growth trend
